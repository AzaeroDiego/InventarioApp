🚀 Inicio Rápido

1

Activar Entorno Virtual

Abre terminal en la carpeta InventarioApp y ejecuta:

python -m venv venv
venv\Scripts\activate
2

Instalar Dependencias

Asegúrate de que el entorno virtual está activo:

pip install -r requirements.txt
3

Inicializar Base de Datos

Crea las tablas e inserta datos iniciales:

flask init-db
4

Ejecutar la Aplicación

Inicia el servidor de desarrollo:

python run.py

📋 Uso Principal de la Aplicación

1. Dashboard

Al ingresar, verás el panel ejecutivo con métricas clave: total de productos, bajo stock, valor del inventario y alertas pendientes.

2. Gestionar Productos

  • Crea nuevos productos con foto e información
  • Busca por nombre o SKU
  • Filtra por categoría o estado
  • Genera códigos QR automáticamente

3. Control de Stock

  • Registra entradas (compras, devoluciones)
  • Registra salidas (ventas, pérdidas)
  • Visualiza historial completo
  • Sistema previene sobreventa

4. Alertas

  • Alertas automáticas cuando stock es bajo
  • Visualización en sidebar con contador
  • Acceso rápido a registrar entrada

5. Reportes

  • Gráficos de productos más vendidos
  • Valor por categoría
  • Movimientos de últimos 30 días
  • Exportar a CSV para análisis
