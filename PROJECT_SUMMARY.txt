✅ 1. AUTENTICACIÓN SEGURA

   • Contraseñas encriptadas ✓
   • Sesiones seguras incluidas

✅ 2. INTERFAZ MODERNA Y RESPONSIVA
   • Diseño mobile-first responsivo
   • Sidebar navegable en todas las secciones
   • Paleta de colores profesional (Verde/Azul)
   • Iconos Font Awesome integrados
   • Layout de 2 columnas adaptable
   • Interfaz completamente en ESPAÑOL

✅ 3. DASHBOARD EJECUTIVO
   • Total de productos activos
   • Productos con bajo stock (alertas)
   • Valor total del inventario ($)
   • Contador de alertas no leídas
   • Historial de actividad reciente
   • Estadísticas de entradas/salidas (7 días)

✅ 4. GESTIÓN COMPLETA DE PRODUCTOS
   ⦿ CRUD Completo (Create, Read, Update, Delete)

   • Listar con buscador + filtros
   • Desactivar (no eliminar) productos
   • Imágenes subidas a /static/uploads/
   • Códigos QR generados automáticamente

✅ 5. GESTIÓN DE CATEGORÍAS
   • crear nuevas categorías
   • Asignar color personalizado a cada una
   • Editar nombre y color
   • Ver cantidad de productos por categoría
   • Categorías iniciales incluidas

✅ 6. CONTROL DE STOCK AVANZADO
   • ENTRADA: Compra, devolución, ajuste, producción
   • SALIDA: Venta, pérdida, ajuste, devolución a proveedor
   • Validación: no vender más del stock
   • Historial completo de movimientos
   • Filtros por tipo y producto
   • Fecha/Hora automática registrada
   • Usuario registrado en cada movimiento

✅ 7. SISTEMA DE ALERTAS
   • Alertas automáticas de bajo stock
   • Alertas creadas al hacer salida si stock < mínimo
   • Contador de alertas sin leer en sidebar
   • Vista de alertas con acceso rápido a entrada
   • Notificaciones en tiempo real (cada 30 seg)

✅ 8. CÓDIGOS QR
   • Generación automática al crear producto
   • Código contiene: SKU, nombre, ID
   • Ver código desde lista de productos
   • Descargar código QR como PNG
   • Imprimir código directamente
   • Almacenado en /static/qrcodes/

✅ 9. REPORTES Y GRÁFICOS
   • Productos más vendidos (gráfico dona)
   • Valor por categoría (gráfico barras)
   • Movimientos últimos 30 días (gráfico línea)
   • Exportar productos a CSV
   • Exportar movimientos a CSV
   • Chart.js integrado para gráficos

✅ 10. API REST IMPLEMENTADA
   • GET /api/alertas/no-leidas - JSON alertas
   • GET /api/producto/<id>/qr - JSON código QR
   • GET /api/reportes/exportar-csv - Exportar datos
   • GET /api/estadisticas - Estadísticas JSON

✅ 11. BASE DE DATOS ROBUSTA
   • SQLAlchemy ORM
   • SQLite (ideal para desarrollo)
   • Tablas: usuarios, categorias, productos, movimientos, alertas
   • Relaciones FK establecidas
   • Índices en campos críticos (SKU, email, fecha)

✅ 12. DATOS INICIALES INCLUIDOS

🚀 CÓMO COMENZAR:

1. Instalar dependencias:
   $ pip install -r requirements.txt

2. Inicializar base de datos:
   $ python run.py
   > flask init-db

3. Crear usuario de prueba:
   > flask create-user
   Email: demo@example.com
   Nombre: Demo User
   Contraseña: 123456

4. Ejecutar servidor:
   $ python run.py

5. Abrir navegador:
   http://localhost:5000
