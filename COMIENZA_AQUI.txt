🚀 INICIO INMEDIATO (5 MINUTOS)
═════════════════════════════════════════════════════════════════════════════

1. Abre PowerShell o CMD
2. Navega a la carpeta:
   cd C:\Users\azaer\Documents\Diego\Bering\InventarioApp

3. Activa el entorno:
   python -m venv venv
   venv\Scripts\activate

4. Instala dependencias:
   pip install -r requirements.txt

5. Inicializa la BD:
   flask init-db

6. Ejecuta la app:
   python run.py

7. Abre navegador (NO USES LIVE SERVER):
   http://localhost:5000

   Email: demo@example.com
   Password: 123456

⚠️  IMPORTANTE: NO uses la extensión "Live Server" de VS Code
    Esta es una aplicación Flask, no HTML estático
